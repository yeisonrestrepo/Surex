Exercise 1. You have to develop an AngularJS directive called 'canadaPhone', and its equivalent for Angular 11.

Requirements

  • It should be applied to text inputs as follows: 
          <input type="text" ng-model="phone" canada-phone />
  • It should format the value of the input with the canadian phone number format "(XXX) XXX-XXXX". This means that the user should type in only numbers, and the directive should add the parentheses and hyphen symbols.
  • It should validate that the phone is correct. The phone is correct when it contains 10 digits.
  • The formatting should be visual only. The ng-model of the directive should ONLY contain numbers, without symbols or spaces.

Examples:

  • The user types "9876543210". The input text should show "(987) 654-3210". The model value should be 9876543210.
  • The user types "987654321". The input text should show "(987) 654-321". This value should be validated as invalid as it only contains 9 characters, and the model value should be null.

Nice to have

The canadian phone numbers can't start by 1, as it could be interpreted as the US international prefix. If the user types a 1 as the first character, it should be ignored.

Examples:

  • The user types "1234567890". The input text should show "(234) 567-890". This value should be validated as invalid as it only contains 9 characters, and the model value should be null.
  • The user types "12345678901". The input should show "(234) 567-8901". The model value should be 2345678901.

Solution:

{{ phoneNumber }}
