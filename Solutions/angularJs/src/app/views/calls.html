Exercise 2. There are multiple resources to be updated and/or removed using HTTP requests. The requests should be executed in a certain order, otherwise the server will refuse some of them.

The Resource class has the following functions in its prototype, $update (PUT http) and $delete (DELETE http). Both functions return a promise.

The execution order consists on executing first all the resources to be deleted. Then, once all the previous promises are resolved, execute all the resources to be updated. In case of any promise has failed then output a message with console log.

Having the following structures: 

        
            // The objects on every array are Resource types 
            const scheduledForDeletion = [resource1, resource2]; 
            const scheduledForUpdate = [resource3, resource4];

Please open the browser developer tools to see the log info.
